## âš™ï¸ Environment & Modes

This project can run in two modes:

- Connected mode (with Supabase persistence)
- Demo mode (no persistence; safe for local demos)

Copy `.env.example` to `.env` and fill in your Supabase credentials to enable connected mode.

Required variables (Vite requires the `VITE_` prefix):

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

If these are not set, the app runs in demo mode. Supabase initialization is gated to avoid crashes, booking/thread/message writes fall back to local ephemeral storage, and realtime is disabled.
